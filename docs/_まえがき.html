まえがき

このチュートリアルについて

このチュートリアルは、UMLモデリングの基礎を学びながら、 実際にWebアプリケーションを開発することを目的としています。

題材として、ボーリングのスコア計算システムを取り上げ、 以下の流れで開発を進めます。

  1. モデルで設計する（クラス図、状態機械図など）

  2. モデルからコードを生成する

  3. Webアプリケーションとして実装する

  4. データベースで永続化する

元のチュートリアルとの関係

このチュートリアルは、『UML初学者向けチュートリアル: モデルを使って ソフトウェアを開発しよう』のWeb版です。

元のチュートリアルでは、コマンドライン版のアプリケーションを開発しました。 このWeb版では、同じビジネスロジックを活用しながら、Webアプリケーション として実装する方法を学びます。

重要なポイント

  • ビジネスロジック（スコア計算）は共通

  • モデル駆動開発のアプローチは共通

  • Webアプリケーション特有の設計を追加

対象読者

このチュートリアルは、以下のような方を対象としています。

  • プログラミングの基礎知識がある

  • Rubyの基本的な文法を理解している

  • Webアプリケーションを作ってみたい

  • UMLモデリングを実践的に学びたい

必要な環境

このチュートリアルを進めるには、以下の環境が必要です。

モデリングツール

astah* Professional

UMLモデリングツールです。クラス図、シーケンス図、ER図などを作成します。 学生・教員の方は無償ライセンスを申請できます。

プログラミング環境

Ruby 3.4.4以上

プログラミング言語です。rbenvを使った管理を推奨します。

Sinatra 4.0

軽量なWebフレームワークです。Bundlerを使ってインストールします。

SQLite 1.7

ファイルベースのデータベースです。開発環境に適しています。

動作確認環境

Webブラウザ

Google Chrome、Safari、Firefox、Edgeなど、モダンなブラウザであれば どれでも動作します。開発者ツールが使えるブラウザを推奨します。

ターミナル（コマンドライン）

Rubyプログラムの実行、Webサーバーの起動などに使用します。 macOSならTerminal.app、WindowsならWindows Terminalなど。

テキストエディタ

プログラムコードを編集するためのエディタです。 Visual Studio Code、Sublime Text、Atomなどを推奨します。

オペレーティングシステム

以下のいずれかのOSで動作を確認しています：

  • macOS 12以降

  • Ubuntu 20.04以降

  • Windows 10/11（WSL2使用）

注記

このチュートリアルのサンプルコードは、主にmacOSとLinuxでの 動作を想定しています。Windowsで実行する場合は、WSL2（Windows Subsystem for Linux） の使用を推奨します。

ネットワーク環境

インターネット接続

Rubyのgemパッケージをインストールするためにインターネット接続が必要です。 また、参考資料の閲覧にも使用します。

ローカルホスト

開発中のWebアプリケーションは、自分のコンピュータ上で動作します （localhost:4567など）。外部からのアクセスは不要です。

詳しいインストール手順は、第1章「準備」を参照してください。

このチュートリアルの構成

第1章 準備

開発環境とモデリングツールを準備します。

第2章 構造の設計

クラス図でシステムの構造を設計します。

第3章 振る舞いの設計

状態機械図などで振る舞いを設計します。

第4章 モデルからコードへの設計

モデルをRubyコードに変換します。

第5章 プログラムのテスト

作成したプログラムをテストします。

第6章 Web化の設計

Webアプリケーションのアーキテクチャを設計します。

第7章 データベース設計

クラス図からER図、SQLへと変換します。

第8章 Web実装

設計をもとにWebアプリケーションを実装します。

第9章 UI設計

ユーザーインターフェースを設計します。

第10章 まとめ

学んだことを振り返り、発展的な学習へつなげます。
