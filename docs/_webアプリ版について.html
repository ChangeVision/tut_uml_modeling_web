Webアプリ版について

このチュートリアルは、『UML初学者向けチュートリアル: モデルを使ってソフトウェアを開発しよう』のWebアプリ版です。 ここでは、元になったチュートリアルとWebアプリ版の関係を説明します。

Webアプリ版の目標

元になったチュートリアルでは、UMLモデリングの基礎を学びながら、UMLモデルからRubyでコマンドラインアプリケーションを作ります。 それに対して、このチュートリアルは、Webアプリケーションを開発することを目的としています。

題材は、元になったチュートリアルと同じボーリングのスコア計算システムです。

  1. モデルで設計する（クラス図、状態機械図など）

  2. モデルからコードを得る方法を規定する

  3. Webアプリケーションとして実装する

  4. データベースで永続化する

このWeb版では、もとのチュートリアルで作成したのと同じスコア計算方法を活用しながら、Webアプリケーションとして実装する方法を学びます。

重要なポイント

  • ビジネスロジック（スコア計算）は共通

  • モデル駆動開発のアプローチは共通

  • Webアプリケーション特有の設計を追加

対象読者

このチュートリアルは、以下のような方を対象としています。

  • プログラミングの基礎知識がある

  • Rubyの基本的な文法を理解している

  • Webアプリケーションを作ってみたい

  • UMLモデリングを実践的に学びたい
